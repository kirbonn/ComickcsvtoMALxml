
# Comick CSV to MAL XML Converter

converts a [Comick](https://comick.io/) csv into an xml that can be imported into [MAL](https://myanimelist.com/) or [Anilist](https://anilist.co/).

dependencies:

```sh
pip install pandas
```
and python 3.x
